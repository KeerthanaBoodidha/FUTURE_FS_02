# 🚀 Lead Management CRM System

## 📌 Project Overview
This project is a Lead Management CRM (Customer Relationship Management) system built using Node.js, Express, and MySQL.  
It helps businesses manage customer leads efficiently by tracking lead details, status updates, and follow-ups.

---

## 🛠️ Technologies Used

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MySQL
- Version Control: Git & GitHub

---

## ✨ Features

✔ Add new leads (Name, Email, Source, Status)  
✔ Update lead status (New / Contacted / Converted)  
✔ Delete leads  
✔ Persistent data storage using MySQL  
✔ REST API integration  
✔ Environment variable configuration using dotenv
